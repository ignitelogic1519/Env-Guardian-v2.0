Capabilities

A complete zero-trust toolkit

Eight enforcement layers that activate automatically inside your restricted zone — and stand down the moment the device leaves.

Geofenced zones

Policy that knows where it is

Draw any restricted area as a map polygon. The app continuously checks GPS against it and flips into enforcement only inside the boundary — so an employee's personal time stays completely private.

  • ✔ Unlimited custom polygon zones
  • ✔ Live "inside / outside" detection with on-map badge
  • ✔ Auto safe-zone: full freedom outside the perimeter
Zero-trust app blocking

Only approved apps run inside

A native on-device enforcer watches the foreground. Anything not on the allow-list is instantly sent back to the home screen. Admins manage the whitelist per-device or org-wide — from the database or the in-app Armory vault.

  • ✔ Global + per-device whitelists (union enforced)
  • ✔ "VIP Allowed" vs "Blocked by Zero Trust" at a glance
  • ✔ Anti-tamper: blocks attempts to force-stop the app
Presence authentication & time-in-zone

Prove you're really there

Inside the zone, the employee scans a physical QR code to authenticate presence — static, or a rotating time-based (TOTP) code that can't be re-used from a photo. A live timer then tracks time-in-zone, on screen and in the notification.

  • ✔ Static or rotating (TOTP) QR — your choice
  • ✔ Live "time in zone" timer with clean glass UI
  • ✔ Pre-scan gate: close background apps before authenticating
Live compliance & telemetry

See every decision, as it happens

A terminal-style perimeter log records every allow/block in real time. Devices heartbeat their location, compliance score, Android version and app usage back to your server — with remote lock available at any moment.

  • ✔ Real-time allow/block ledger
  • ✔ Compliance scoring + remote admin lock
  • ✔ Per-app usage & device inventory reporting
Also included

More enforcement, more control

🌐

Network guard (VPN)

An always-on, no-root VPN cuts internet to non-whitelisted apps in-zone and switches off automatically the instant the device leaves. Disable attempts are detected and reported.

⏱️

Per-app time limits

Grant an app a daily budget (e.g. 30 min). Usage is measured on-device; once spent, the app is blocked for the rest of the day.

🚫

Anti-theft binding

Each device is permanently bound to its first owner. A lost or stolen phone can't be re-registered under a new identity.

🔔

Auto-foreground on entry

Cross into the zone and the app pops forward with a full-screen prompt to authenticate — no hunting for the icon.

🔒

Per-device tokens

Every device gets its own auth token at registration, layered on top of the API key for stronger request security.

📱

OEM keep-alive

Guided auto-start & battery setup for Xiaomi, Oppo, Realme, Vivo and Samsung so the monitor keeps running.
